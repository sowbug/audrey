Improvements in Audrey Software Release 1.2

Congratulations on your purchase of Audrey, the first Household Digital
Assistant (HDA)!

One of Audrey's many advantages is its ability to automatically update its software. We will continue working hard to improve Audrey's software so that your experience becomes even greater over time.

We recently completed an update to Audrey's software that includes the following list of improvements. We hope these changes help make your Audrey experience more enjoyable than ever before!

Thanks,

The Audrey Team @ 3Com

  • Significantly improved Internet connectivity, including simplified Internet Service Provider (ISP) setup and additional support for smaller, regional ISPs.
  • Support for Web sites with daughter windows has been added.
  • Added alarm notifications for synced Palm events in the Datebook.
  • Improved SMTP email support.
  • Modem dial-up sounds can now be enabled (optional). To enable or disable modem dial-up sounds, simply launch the Actions Palette, tap "Audrey options," tap "Internet Settings," tap "ISP dialing," and then check or uncheck the "Enable Modem Speaker" check box.
  • Expanded support for large (> 80K) email messages.
  • The MAC address for attached USB/Ethernet adapters is now listed in the Audrey profile screen to aid troubleshooting.
  • Enhanced Channel performance for faster and more stable downloads.
  • More Channels are available and can be added simply by tapping "Channel Select" in Audrey Options.
  • Improved printing capabilities and resolution with select Canon printers. *
  • Quicker and more reliable syncing to PalmTM and HandspringTM Visor handheld organizers, plus added support for the Sony CLIÉTM.
  • General improvements in system and memory performance.

* Canon printer models BJC-85, BJC-85W, BJC-2100.
