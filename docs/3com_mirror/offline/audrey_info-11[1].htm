Actions Palette

On the "Actions Palette" screen next to the connect icon, the connection speed is displayed when Audrey is connected. If Audrey is not connected, a "Not Connected" message will be next to the disconnect icon. When you touch the Audrey options icon, the menu offers 13 choices of options. One choice is a Memory icon that will display how much memory Audrey has consumed and how much memory is left. These two items, connection speed and memory, can provide reliable information when troubleshooting any problems with Audrey.

If the Actions palette button is pressed while running these applications -- datebook, mail, channel, or browser -- the Actions Palette" screen will display datebook options, mail options, channel options, or browser options icons respectively.

Cut, Copy, and Paste: All the basic editing features are available on the "Actions Palette" screen. Highlight text by tapping and dragging the stylus (without losing contact with the screen) over text that you wish to move or copy. After the text is highlighted, press the Actions Palette button and tap the:

  • cut icon if you wish to move the text
  • copy icon if you wish to copy the text

and the text will be placed on Audrey's internal clipboard. At this point, go to the application and exact location where you wish the text to be placed. Tap that location to place the cursor there. Press the Actions palette button and tap the Paste icon. All the material on the internal clipboard will be pasted where the cursor is located.

Snapshot: Make a screen capture from a currently open application and store it in the address book. Launch the application that has the screen you wish to capture. Press the Actions Palette button, tap snapshot icon, and tap okay to continue. The "Actions Palette" screen disappears and the application you wish to capture returns. Using the stylus, tap to the left and above of the area to capture and drag the stylus (never losing contact with the screen) to the right and below the area to capture. A box will appear around the area selected. Lift the stylus. The Action Palette launches again displaying what was captured. Tap okay to add this image to your snapshots archive in the Address Book application. (You also have the choice to cancel or try again). When the address book opens, title the snapshot, add a note about the image, and select the following options: add to Address Book, use in e-mail message, or both. If it is added to an e-mail, it will be attached as a file.

Print: The "Actions Palette" screen also has a print icon. This icon is grayed out unless Audrey has a USB printer connected. Audrey only has print drivers for a Canon BJC-85 or a Cannon BJC-1200. These are the only two printers compatible with Audrey at this time. When one of these printers is connected to one of the USB ports, the icon is active and a tap this icon will produce a printed page if the printer is turned on. Open the application and screen you wish to print. Press the Actions Palette button and tap print icon.

TIP: Tap the center of the screen that you wish to print before opening the Actions Palette and tapping the print icon.
