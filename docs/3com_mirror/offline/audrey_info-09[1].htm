Browser

Audrey's web browser supports:

  • HTML 3.2
  • Flash 4
  • Real Audio
  • Real Player G2 version 4.0
  • Java Script 1.1
  • SSL (Secure Socket Layer) 128 bit Encryption - for secure online transactions

The address book will store all the bookmarks (favorite sites marked) that are entered or chosen.

Changing browser configuration or options is a three-step process. While the Browser is running, press the Actions Palette button, tap the browser options icon to the left of Audrey options, and the browser options screen offers the following configuration options:

  • Set the home page
  • Accept or reject cookies
  • Clear or keep cache
  • Increase or decrease the size of the fonts for downloaded text

Audrey's web browser does not support file download features because there is no hard drive to store large amounts of data. There will be no response from Audrey if you tap a download feature on a web page.

If Audrey encounters an unsupported or incompatible file or format, the following will be displayed:

This web site contains content which Audrey does not currently support.

Some web sites require Internet Explorer or Netscape Navigator for viewing all their functions and features. When those sites detect a different browser, the page will not display or will prompt the download of one of the listed browsers. Audrey displays no message or acknowledgement of any problem with these sites.

Audrey terminates analog web browser connections automatically when no activity occurs for a 15-minute period. Inactivity means that Audrey is not surfing the web, playing streaming audio files, or interacting on the web. This automatic timeout of 15 minutes cannot be changed.
