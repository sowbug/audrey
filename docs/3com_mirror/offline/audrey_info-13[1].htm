e-mail

If this is the first time mail is being sent and you are unable to send mail, check the mail options to ensure "Disable Email" is not checked. Press the Actions palette button while in the mail application, tap Mail options, tap Server Settings, and confirm that "Disable Email" does not have an x in the box. This should also be confirmed if you are unable to receive mail.

While in the mail application, press the Actions palette button, tap Mail options, and check the Username and password to ensure they were entered correctly. Verify their accuracy by retyping the information. Check the information in the Incoming (POP3) and Outgoing SMTP fields with the ISP to validate their accuracy.

If you are unable to send mail and have sent mail in the past, check your "Sending Later" view to see if the mail is listed there and tap Send now. If that is not the problem, then perhaps Audrey was offline when you tapped Send and is waiting for the next connection or download to the ISP. To force the mail to be sent now, tap the Get New Mail icon.

Audrey can only support these extensions in mail attachments:

  • .wav for voice messages,
  • .gif for scribble messages and snapshots,
  • .vcf for address book records.

Only one of these formats can be used per e-mail. If the mail is going from one Audrey to another, a voice message or scribble message will be displayed in the body of the message. If the mail goes from an Audrey to another device such as a personal computer, the voice or scribble message will be an attachment. If one Audrey sends another Audrey a message containing address book record(s), the .vcf record(s) will be imported into the receiving Audrey's address book.

Audrey does not support word or other word processing programs such as notepad so it will not recognize attachments with those application extensions. If mail has .doc, .txt, .exe, .bmp or other unsupported extensions for attachments, do not delete the message but wait until you can view the attachment from a personal computer system.

Do not delete mail from Audrey just because you cannot read it. Save it and view the mail from another device such as a personal computer. If e-mail is deleted from Audrey, the next download will update the mail server and eliminate the messages deleted on Audrey.

If the inbox on Audrey is full or has very little space left, Audrey will issue a message:

Audrey can't download new messages because your mailbox is full. Delete unwanted messages

The message will appear before downloading the incoming mail. If any of the mail in a given session fits into the space left, it will download that mail but not the others that don't fit.

  • Audrey has 1MB of storage for the Inbox. Each message has an upper limit of 400K. An average e-mail is approximately 2 to 3K; voice messages are about 150K for each minute.
  • The Outbox stores 512K and will hold a maximum of 10 e-mails.

When user prepares a message to send and the message exceeds the 512k size, Audrey displays following message:

This email message cannot be sent because it is too large

When a message makes the outbox cross its 512K limit:

  • If there are e-mails in "sent" state, they will be deleted on oldest-one-first basis until there is enough room to fit this new mail in outbox.
  • If there are messages in "Send Later" state that cannot be deleted, Audrey displays the following message:

      This email message cannot be sent until email messages in the 'sending later' list are sent
