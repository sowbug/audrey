How do I sync with the Address Book?

1 Launch the Address Book.
Press the Address Book button on the console. Then tap the email or phone icon. Make sure the Address Book is open in the drop-down menu.

2 Find the e-mail address or phone number you wish to "sync."
Once you've located the entry, tap the person's name.

    NOTE: This method can also be used to send a user's Address Book entries to the Address Book application on Audrey.

3 Tap inside the Sync To: drop-down menu.
Tap the name of the Address Book to which you would like to sync the entry. Then tap done. The entry will be synced to that device upon the next HotSync® operation.

4 The next time you sync
this device with Audrey, the event will be sent to the Palm Address Book.
